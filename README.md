# FPGA Access Control System

## Description
This project implements a secure access control system using SHA-256 hashing on FPGA (Nexys A7).

## Folder Structure
- project_files: All custom Verilog modules
- third_party: SHA-256 hashing modules
- testbench: Simulation and verification files
- constraints: FPGA pin configuration
- report_resources: Images and diagrams used in report
